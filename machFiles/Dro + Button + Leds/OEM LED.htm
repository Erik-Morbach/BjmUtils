OEM LEDs

From MachCustomizeWiki

This list gives the codes to be used in calls of SetOEMLED and GetOEMLED to access all LEDs.

If you are using a version of Mach3 prior to 1.90 then to access data in this list that are in the range 800 upwards you will need to refer to the deprecated calls (e.g. SetLED) and corresponding codes.
